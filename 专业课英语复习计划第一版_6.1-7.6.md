# 6/1-7/6 专业课 + 四级英语复习计划 第一版（物理加强）

目标：每一科都尽量 4.0+。你确认 4.5 对应 93 分，所以本计划按“85+ 保底、90+ 稳、93 冲高绩点”来排。  
核心策略：专业课为中心，6/13 前英语每天不断线；物理是风险科，先保 85+，再争 90+；C++、数电这类可控科目优先冲 93。

## 0. 依据和默认范围

## 1. 成绩目标对应策略

你目标是每一科都 4.0+，所以不能只救弱科，也不能只刷强科。现在按“保底线 + 冲刺线”排：

| 科目 | 保底目标 | 冲刺目标 | 当前策略 |
|---|---:|---:|---|
| 大学物理上 | 85+ | 90+ | 风险最高，每天碰，周末重练旧作业和模型题 |
| 高等数学下 | 88+ | 93 | 高权重科目，持续刷题，重积分/曲线曲面积分/级数是提分关键 |
| 数字电路 | 90+ | 93+ | 题型规律强，适合冲 4.5 |
| 计算机系统概论 | 88+ | 93 | LC-3、补码、寻址、I/O 稳住后可以冲高 |
| C++ 综合实训 | 90+ | 93+ | 最可控，靠题单完成度、代码质量、能讲清楚来冲 |
| 英语四级 | 稳过 | 不作为绩点主线 | 6/13 前每天保底，不抢专业课主时间 |

各科一句话策略：

- 物理：先把每类题的模型补起来，避免低于 85，再争 90+。
- 高数：保持每天计算手感，靠题型熟练度冲 90+，模拟稳定后再冲 93。
- 数电：尽量冲 93，因为真值表、卡诺图、状态机都能训练到稳定。
- 系统：概念链不能散，LC-3 和汇编题要能手写。
- C++：能跑、能交、能解释，争取做成冲 93 的科目。

物理 85+ 到 90+ 的具体打法：

1. 每天至少碰一次物理，哪怕只有 30 分钟。
2. 先补“模型”，再做题：看懂题目属于哪个模型，比背公式更重要。
3. 旧作业优先级高于新题，因为期末常从作业模型变形。
4. 每道错题必须写错因：不会建模、公式选错、方向符号错、积分/代数错。
5. 每周至少做一次“闭卷默写公式 + 典型模型”。
6. 每周末做一次物理小测：力学/电磁学各 30-45 分钟，按考试状态限时。

冲 4.5 的具体打法：

- C++：题单不能只过样例，要补边界输入；每题都要能说出思路、易错点、测试方法。
- 数电：每类题至少做到“看到题型知道流程”：化简、组合设计、触发器转换、状态机设计。
- 高数：如果 6/30 前模拟能稳定 85+，考前周追加冲 93 的限时综合题。
- 系统：如果 LC-3 汇编和寻址方式能稳定，考前周追加综合题冲 90+。

## 3. 各科复习抓手

### 大学物理上：重点加强

按 85+ 要求，不只“看懂”，要能独立做题。

力学重点：

- 质点运动学：位移、速度、加速度、圆周运动。
- 牛顿定律：受力分析、约束、摩擦、斜面、绳。
- 功和能：动能定理、机械能守恒、非保守力做功。
- 动量：冲量、动量守恒、碰撞、质心。
- 刚体转动：转动惯量、力矩、角动量、转动定律、滚动。

电磁学重点：

- 静电场：库仑定律、电场强度、叠加、高斯定理。
- 电势：电势能、电势、电场与电势关系。
- 导体、电容、电介质。
- 恒定电流：电流密度、电动势。
- 磁场：洛伦兹力、安培力、毕奥萨伐尔、安培环路。
- 电磁感应：法拉第定律、楞次定律、动生/感生电动势。

每次物理学习固定流程：

```text
10min 公式和模型
30-60min 旧作业/典型题
10min 写错因
```

### 高等数学下

重点：

- 空间解析几何。
- 多元函数微分学：偏导、全微分、链式法则、隐函数、极值。
- 重积分：二重、三重、换序、极坐标、柱坐标、球坐标。
- 曲线曲面积分：如果老师讲过 Green/Gauss/Stokes 就重点练。
- 无穷级数：判敛、幂级数、Taylor 展开。

### 数字电路

重点：

- 数制、编码、逻辑代数。
- 卡诺图。
- 组合逻辑：编码器、译码器、数据选择器、加法器、比较器。
- 触发器。
- 时序逻辑：状态表、状态图、计数器、寄存器、序列检测器。

### 计算机系统概论

重点：

- 位、补码、浮点数。
- 逻辑门、组合逻辑、存储元件。
- 冯诺依曼模型、指令周期。
- LC-3 指令、寻址方式、汇编。
- 子程序、栈、TRAP、I/O。
- C 与机器层：数组、指针、函数调用。

### C++ 程序设计综合实训

按老师题单推进。每次都要跑代码。

每次记录：

```text
题号/题目名：
是否跑通：
卡住的 bug：
下一步：
```

## 8. 调整规则

- 如果任意科模拟题低于 85：接下来 3 天每天给该科至少 60-90min。
- 如果物理模拟题低于 80：接下来 3 天每天给物理至少 90min，并减少系统或 C++ 30min。
- 如果某科模拟稳定 90+：该科不再大量刷题，改成每天 20-30min 保温。
- 如果某科有机会冲 93：考前周追加 1-2 次限时综合题。
- 如果高数明显崩：周末从 C++ 或系统里挪 30-60min 给高数。
- 如果 C++ 题单难度高：每周五下午或周末给 C++ 加 30min，但不挤掉物理。
- 如果周五晚上想放松：直接放松，不补任务。
- 如果某天状态很差：只做物理 45min + 英语/最近考试 30min。
