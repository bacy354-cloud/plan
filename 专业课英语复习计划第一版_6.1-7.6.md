当前目标：每一科尽量 4.0+；可控科目冲 4.5，也就是 93 分。

## 0. 这版的新规则

你今天试跑后反馈了几个关键点，所以这版按下面规则重排：

1. 每天预留约 45min 给实验报告、平时作业、老师临时任务等非复习工作。
2. 6/13 前英语每天 1h。
3. 高数 + 物理是主线，正常日合计保底 3h15min。
4. C++ 不是系统复习课，而是敲代码做题；需要的日子给 45-60min。
5. 6/13 前不系统复习数电和计系，只做作业、看题目、标记不会点。
6. 周五晚上继续默认放松。
7. 周六日是两天合计 8h，不是每天 8h。
8. 高数当前进度按 6/4 已复习到“偏导数”处理，后面数学节奏稍微加快。

有一个现实冲突要先写清楚：

- 周四只有 1.5h，不可能同时完成英语 1h、作业 45min、物理高数 3h15min。
- 周五基础时间 4.5h，接近但仍略紧；所以周五设为弹性日，可以按实际情况加 30-60min。
- 周末单日 4h，也塞不下这三项。

所以执行优先级是：

1. 正常日预留非复习作业 45min，防止实验报告和平时作业拖炸；周四保温日不强制。
2. 6/13 前英语尽量 1h，短时间日压缩到 30min。
3. 物理 + 高数尽量靠近 3h15min，正常日严格做到。
4. C++ 只在有空间或题目需要时安排。

如果你之后想严格做到“每天物理+高数 3h15min”，周末两天合计时间至少要从 8h 提到 10h。

## 3. 高数两本习题使用方法

你现在有两本高数习题：

- 小本：宋浩复习课配套题，题少，适合看完一个知识点后立刻做。
- 大本：题多，适合晚上或弹性时间集中刷题。

具体用法：

| 习题本 | 用法 | 每次时间 | 每次题量 |
|---|---|---:|---|
| 小本 | 当作课后作业，学完一个点马上做 | 15-25min | 4-8 题 |
| 大本 | 当作主刷题本，隔天或周五/周末集中刷 | 40-60min | 6-12 题 |

高数每次学习拆成三步：

```text
1. 复习/看课：30-45min
2. 小本即时巩固：15-25min
3. 大本刷题：不是每天都做，有安排时做 40-60min
```

大本刷题规则：

- 不要求每天刷。当天没时间可以跳过。
- 如果一天没刷，第二天或周五弹性时间补一组，不要硬补到爆炸。
- 补刷最多 60min，超过就停。
- 每组先做基础题和中档题，难题最多 1-2 道。
- 一道题卡 10min 还没思路，就标记原因，先过。
- 大本优先刷“刚学的新知识 + 前一个知识点错题”，不要只刷新内容。

从 6/4 起，高数进度按这个顺序推进：

```text
偏导数
-> 全微分
-> 链式法则
-> 隐函数求导
-> 多元函数极值
-> 二重积分
-> 三重积分
-> 曲线曲面积分
-> 无穷级数
```

## 8. 调整规则

- 如果非复习作业超过 45min，先压缩 C++ 或数电/计系，不压缩物理。
- 如果物理当天完全没碰，第二天第一块先补物理 45min。
- 如果高数连续两天没做题，第三天必须先做高数计算题。
- 如果小本当天没做，不补；如果大本当天没刷，可以挪到下一个周五/周末补一组。
- 如果大本刷题正确率低于 50%，下一次先回小本和例题，不继续硬刷。
- 如果 C++ 老师突然催题，选周二或周末挪 45-60min 做代码。
- 周四是保温日，不补债；如果很累，只做 30min 英语/公式/错题也算完成。
- 周五是弹性日；如果实验报告、C++ 或作业堆了，优先放到周五，不挤掉物理。
- 不补债，计划是帮你稳住节奏，不是制造压力。
